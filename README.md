# 🏎️ Dashboard de Análise de Vendas da Porsche

Este projeto consiste em um dashboard interativo para análise de vendas da Porsche, integrando **Excel avançado** e **Inteligência Artificial** para a geração automatizada de relatórios e insights estratégicos. 

Projeto desenvolvido durante a **Aceleração AI Reports**.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Excel Avançado:** Estruturação dos dados, tabelas dinâmicas e construção dos visuais do dashboard.
* **GPT Agents:** Automação da análise descritiva e geração de insights de negócios.
* **Claude Code:** Suporte na engenharia de dados, validação de lógica e estruturação dos relatórios.

---

## 📌 Principais Funcionalidades & Insights
* **Visão Geral de Vendas:** Acompanhamento de faturamento, volume de modelos vendidos e ticket médio.
* **Análise Inteligente por IA:** Relatórios gerados via agentes para identificar tendências de mercado e modelos de maior desempenho.
* **Tomada de Decisão:** Visualização clara para suporte a estratégias comerciais.
